# Retwisters (iOS Portfolio App)

Retwisters is a personal iOS development project used to practice and demonstrate mobile app development skills.

## Overview

This project focuses on:

- Building user interfaces with **SwiftUI**
- Managing view state and navigation
- Integrating with a **Firebase** backend
- Working with asynchronous data loading and real-world app structure

The app is currently in active development and used as a portfolio project to showcase code quality, organization, and modern iOS patterns.

## Tech Stack

- **Language:** Swift
- **Framework:** SwiftUI
- **Backend:** Firebase (Auth & Firestore)
- **Tools:** Xcode, Git, GitHub

## Status

This is an in-progress personal project. The codebase and features are continually being refined as I learn and improve as a developer.
